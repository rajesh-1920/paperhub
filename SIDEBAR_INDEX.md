## 🔑 Key Concepts

### Core Concepts

- **Fixed Desktop Sidebar** - Always visible on desktop
- **Mobile Overlay** - Slide-in drawer on mobile
- **Role-Based Access** - Show/hide items by role
- **Collapse Mode** - Desktop-only icon-only view
- **Active Highlighting** - Automatic current page detection

### Roles

- **ADMIN** - Full system access
- **OFFICER** - Document review & processing
- **USER** - Standard document submission

### Technical Concepts

- **IIFE Pattern** - Modular JavaScript encapsulation
- **Data Attributes** - Role filtering via HTML
- **LocalStorage** - Preference persistence
- **CSS Transitions** - GPU-accelerated animations
- **Semantic HTML** - Accessibility-first markup
