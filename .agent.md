# PaperHub Agent Configuration

This document instructs AI agents on how to work with the PaperHub project.

## Theme Configuration: 🔒 FINAL & LOCKED

**The CSS theme and styling is FINAL as of May 11, 2026.**

### Critical Rule
✅ **CAN modify CSS files** for:
- Aligning pages with dashboard theme
- Fixing styling inconsistencies
- Improving component layouts
- Adding missing animations/transitions

🚫 **DO NOT modify core dashboard theme**:
- Primary colors (`#0284c7`, `#06b6d4`, `#0f172a`)
- Dashboard gradient patterns (135deg)
- Role-based color schemes (admin/officer/user)
- Core shadow and blur effects on dashboard
- Typography hierarchy on dashboard pages

**Rule**: Any page that doesn't align with the dashboard should be updated to match the main theme.

### Color Reference (For Non-CSS Work)
When working on features, refer to these colors for consistency:
- Primary: `#0284c7` (Blue)
- Secondary: `#06b6d4` (Cyan)
- Dark: `#0f172a` (Navy)
- Light: `#f8fafc` (Slate)

## Approved Work Areas

✅ **Safe to modify WITHOUT asking**:
- JavaScript files in `/public/assets/js/`
- HTML content and structure
- Backend/server-side code
- CSS files to align with dashboard theme
- Adding animations/transitions matching dashboard style
- Fixing styling inconsistencies on any page

🚫 **DO NOT modify without explicit user request**:
- Core dashboard theme colors and gradients
- Role-based color schemes (admin teal, officer orange, user blue)
- Primary dashboard shadow and blur effects
- Dashboard typography hierarchy

---

## Workflow Guidelines

1. **Check `.instructions.md` first** - Always read this for dashboard theme specifications
2. **Align all pages** - If any page doesn't match dashboard styling, update it
3. **Preserve dashboard colors** - Don't change primary dashboard theme colors
4. **Use dashboard gradients** - Match 135deg gradient patterns throughout
5. **Apply dashboard animations** - Use same 0.25s cubic-bezier transitions everywhere
6. **Document significant changes** - If you update major styling, note it in comments
