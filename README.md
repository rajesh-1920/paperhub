# PaperHub

PaperHub is a fully static HTML5/CSS3 product-style prototype for professional document operations.

It includes a complete multi-page experience with a redesigned information architecture, role-based
workspace layouts, and a consistent UI system for landing, authentication, dashboard, documents,
and profile workflows.

## Project Structure

- index.html
- pages/login.html
- pages/dashboard.html
- pages/documents.html
- pages/profile.html
- assets/css/variables.css
- assets/css/components.css
- assets/css/styles.css

## Design System

- Professional page structures with clear section hierarchy
- Dedicated app shell patterns for workspace pages
- Responsive desktop/tablet/mobile behavior
- Reusable card, table, stage-board, form, and modal components
- Light and dark mode powered by CSS variables

## Pages

- Landing: product overview, platform pillars, workflow blueprint, CTA
- Login: secure workspace entry with trust and access context
- Dashboard: operational metrics, stage capacity board, queue and activity view
- Documents: register-focused management, lanes, timeline, upload modal
- Profile: identity, permission model, security controls, account activity

## Run

Open index.html directly in any modern browser.
